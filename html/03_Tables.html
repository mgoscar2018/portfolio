Para crear tablas en HTML se utilizan las siguientes etiquetas: table, caption, tr (table row), th (table head) y td (table data).

Código HTML: 
            <table border="1">
                <caption>Título de la tabla</caption>
                <tr>
                    <th>Name</th>
                    <th>Lastname</th>
                    <th>Email</th>
                </tr>
                <tr>
                    <td>Oscar</td>
                    <td>Miranda</td>
                    <td>mgoscar@gmail.com</td>
                </tr>
                <tr>
                    <td>Homer</td>
                    <td>Simpson</td>
                    <td>hsimpson@mail.com</td>
                </tr>
                <tr>
                    <td>Bart</td>
                    <td>Simpson</td>
                    <td>bsimpson@mail.com</td>
                <tr> 
                <tr>
                    <td>Lisa</td>
                    <td>Simpson</td>
                    <td>lsimpson@mail.com</td>
                <tr>                
            </table>
RESULTADO:
Título de la tabla
Name Lastname Email
Oscar Miranda mgoscar@gmail.com
Homer Simpson hsimpson@mail.com
Bart Simpson bsimpson@mail.com
Lisa Simpson lsimpson@mail.com

Con las etiquetas colspan y rowspan podemos fusionar celdas:

Código HTML: 
            <table border="1">
                <caption>Título de la tabla</caption>
                <tr>
                    <th colspan="2">Name</th>
                    <th>Lastname</th>
                    <th>Email</th>
                    <th>Grupo</th>
                </tr>
                <tr>
                    <td>Oscar</td>
                    <td>Miranda</td>
                    <td>mgoscar@gmail.com<td>
                    <td>A<td>
                </tr>
                <tr>
                    <td>Homer</td>
                    <td>Simpson</td>
                    <td>hsimpson@mail.com</td>
                    <td rowspan="2">B<td>
                </tr>
                <tr>
                    <td>Bart</td>
                    <td>Simpson</td>
                    <td>bsimpson@mail.com</td>
                <tr> 
                <tr>
                    <td>Lisa</td>
                    <td>Simpson</td>
                    <td>lsimpson@mail.com</td>
                    <td>C<td>
                <tr>                
            </table>
RESULTADO:
Título de la tabla
Name Email Grupo
Oscar Miranda mgoscar@gmail.com A
Homer Simpson hsimpson@mail.com B
Bart Simpson bsimpson@mail.com
Lisa Simpson lsimpson@mail.com C

Aplicando estilos CSS, podemos mejorar mucho visualmente la información:

Código CSS utilizado:
            table, th, td {
                border: 1px solid black;
                border-collapse: collapse; /* colapsa el borde de la tabla */
                padding: 5px;
            }
            
            caption {
                color: #e63946;
                font-family: 'Grape Nuts'; /*https://fonts.google.com/specimen/Grape+Nuts*/
                font-size: 20px;
                line-height: 20px;
            }
            
            th {
                color: #f1faee;
                background-color: #1d3557;    
            }
            
            /*Renglones pares */
            tr:nth-child(even) {
                background-color: #f1faee;
            }
            
            /*Renglones impares */
            tr:nth-child(odd) {
                background-color: #a8dadc;
            }
RESULTADO:
Título de la tabla
Name Lastname Email
Oscar Miranda mgoscar@gmail.com
Homer Simpson hsimpson@mail.com
Bart Simpson bsimpson@mail.com
Lisa Simpson lsimpson@mail.com
