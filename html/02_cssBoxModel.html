Utilizando la etiqueta <pre> podemos colocar texto pre-formateado, es decir, respeta espacios, tabulaciones, saltos de línea, etc.
Código CSS utilizado:
                p {
                    color: white;
                    background-color: #ff006e;
                    padding: 15px;
                    border: 10px solid #ffafcc;
                    margin: 20px;
                }
Código HTML:
<p>Contenido del elemento</p>

Resultado:

Contenido del elemento

Modificando el padding y cambiando el color de fondo, podemos lograr que un simple link parezca un botón:
Código CSS:
                a {
                    display:inline-block; 
                    text-align: center;
                    text-decoration: none;
                    color: white;
                    padding: 15px 25px;
                }
                
                a:link, a:visited {
                    background-color: #cdb4db;    
                }
                
                a:hover, a:active {
                    background-color: #ffc8dd;
                }
Código HTML:
<a href="01_basic.html">LINK - CLICK</a>

Resultado:
LINK - CLICK
