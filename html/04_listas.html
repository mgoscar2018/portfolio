Se tienen 3 tipos de listas:

Listas Ordenadas <ol> (Ordered list)

            <ol>
                <li>Elemento X</li>
                <li>Elemento Y</li>
                <li>Elemento Z</li>
            </ol>
  1. Elemento X
  2. Elemento Y
  3. Elemento Z

Listas No Ordenadas <ul> (Unordered list)

            <ul>
                <li>Elemento X</li>
                <li>Elemento Y</li>
                <li>Elemento Z</li>
            </ul>
  • Elemento X
  • Elemento Y
  • Elemento Z

Listas de definición <dl> (Description list)

            <dl>
                <dt>Término X</dt>
                <dd>definción X</dd>
                <dt>Término Y</dt>
                <dd>definción Y</dd>
                <dt>Término Z</dt>
                <dd>definción Z</dd>
            </dl>
Término X
definción X
Término Y
definción Y
Término Z
definción Z

Aplicando estilos CSS podemos convertir una lista desordenada como esta:

            <ul>
                <li><a href="01_basic.html">Elementos básicos HTML</a></li>
                <li><a href="02_cssBoxModel.html">Modelo de caja CSS</a></li>
                <li><a href="03_Tables.html">Tablas</a></li>
                <li><a href="04_listas.html">Listas</a></li>
            </ul>
