En este lugar veremos las diferentes maneras en que se pueden identificar los elementos para aplicarles estilos CSS

Con el selector universal "*" cambiamos el tipo de letra (Verdana) a todos los elementos

Este párrafo tiene una clase: clase1, al que se le cambió el tamaño de la letra a 28px.

Se pueden asignar muchas clases a un mismo elemento, este párrafo tiene dos clases: clase1 y clase2.

También se puede utilizar el identificador "id", pero a diferencia de las clases, sólo puede tener un identificar cada elemento, en este párrafo se llama "iden1".

Este párrafo tiene el identificador llamado "iden2".
