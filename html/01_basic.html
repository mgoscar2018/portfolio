Párrafo con diferentes formatos de texto, por ejemplo:
bold <b> vs <strong> visualmente no tienen diferencias, pero en significado el texto con strong es más importante.
Lo mismo ocurre con <i> de Italic vs <em> de Emphasis
adicional, podemos encontrar más formatos como: letras pequeñas <small>, para código <code>, texto remarcado <mark>, suprimido o 'delete' <del>, texto insertado <ins>, texto con subíndice <sub> y superíndice <sup>.

Si se requiere desplegar carácteres especiales como: ® © Ω ⅕ ☆ ⦜ , etc. Dar clic al siguiente enlace para mostrar el listado completo: Tabla de carácteres

Con la etiqueta <style> se puede agregar código CSS de manera individual a cada elemento. Este código nos permite entre muchas cosas cambiar los colores de diferentes maneras:

  1. Por nombre style="border: 10px solid red;"
  2. Por código rgb style="border: 10px solid rgb(111,15,124);"
  3. Por código rgba (con transparencia) style="border: 10px solid rgba rgba(111,15,124,.5);"
  4. Por código hsl style="border: 10px solid hsl(240, 100%, 27%);"
  5. Por código hsla (con transparencia) style="border: 10px solid hsla(240, 100%, 27%,.5);"
  6. Con código hexadecimalstyle="border: 10px solid #008000;". NOTA: si en el código dos o más digitos son iguales, puede ser simplificado a uno solo de la siguiente manera: #008000 por #080
