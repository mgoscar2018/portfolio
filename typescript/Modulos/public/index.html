Trabajando con módulos en TypeScript

Para trabajar con módulos, es necesario instalar webpack, el cual nos permitirá utilizar un enfoque modular en el desarrollo de aplicaciones WEB

  1. En consola escribir npm install webpack webpack-cli webpack-dev-server, al finalizar nuestra carpeta del proyecto tendrá los archivos necesarios para desplegar la aplicación (package.json, package-lock.json, node_modules, etc)
  2. Ahora instalar TypeScript Loader: npm install typescript ts-loader --save-dev para poder cargar nuestras clases.
  3. Actualizar el archivo package.json con el comando: npm init -y
  4. Abrir el archivo package.json y agregar lo siguiente detro de la variable "scripts": "start": "webpack-dev-server --mode development". Esto permitirá ejecutar el comando "start", el cual iniciará nuestro webpack server en el modo de desarrollo. Debe quedar de la siguiente manera: 
    "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "webpack-dev-server --mode development"
},
  5. Crear archivo webpack.config.js y pegar lo siguiente: 
    const path = require('path');

module.exports = {
    entry: path.join(__dirname, '/NOMBRE_ARCHIVO.ts'),
    output: {
        filename: 'NOMBRE_ARCHIVO.js',
        path: __dirname
    },
    module: {
        rules: [
            {
                test: /\.tsx?$/,
                loader: 'ts-loader',
                exclude: /node_modules/,
            },
        ]
    },
    resolve: {
        extensions: [".tsx", ".ts", ".js"]
    },
};
  6. Dentro del archivo webpack.config.js reemplazar NOMBRE_ARCHIVO por el nombre del archivo de entrada que tengamos (para este caso se llama "modulos").
  7. Iniciar nuestro servidor con npm start
  8. Verificar su funcionamiento aquí: LOCALHOST
