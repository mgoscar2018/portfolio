Instalar TypeScript

Para instalar en Windows, se debe hacer lo siguiente:

  1. Abrir terminal como Administrador
  2. Ejecutar el siguiente comando: npm install -g typescript

Compilar archivo TS

Para compilar un programa ts (typescript), desde la consola se escribe el siguiente comando:
tsc NOMBREARCHIVO.ts

NOTA: En caso de obtener un error como el siguiente "No se puede cargar el archivo NOMBRE_ARCHIVO porque la ejecución de scripts está deshabilitada en este sistema. Para obtener más información, consulta el tema about_Execution_Policies..."

  1. Ejecutar windows + R --> gpedit.msc
  2. Ir a Plantillas administrativas > Componentes de Windows > Windows PowerShell
  3. Seleccionar Activar la ejecución de scripts, click derecho, editar
  4. Seleccionar "Habilitada" y "Permitir todos los scripts", Aplicar

Inicializar proyecto TypeScript

Escribiendo en consola el comando tsc -init se inicializa un proyecto creando en automático el archivo tsconfig.json

Compilación en "Watch Mode"

Con el comando tsc -w automáticamente todos los cambios que realicemos en el archivo "ts" serán compilados y actualizados en el archivo "js" resultante.
