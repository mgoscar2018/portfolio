Instalación de Angular

Desde una terminal en modo "administrador" ejecutar el siguiente comando:
npm install -g @angular/cli

Crear aplicación llamada "primerApp"

ng new nombre-aplicacion

Ejecutar aplicación y verificar que está funcionando correctamente

Entrar al directorio que tiene el nombre de nuestra aplicación "primerApp" y ejecutar el siguiente comando:

ng serve -o
Para detener el servidor de Angular, pulsar CTRL-C dentro de la terminal en la que ejecutamos el servidor

Ejecución del servidor desde Visual Studio Code

Al trabajar con Angular, es importante siempre abrir Visual Studio Code con privilegios de Administrador por los comandos que se requieren ejecutar en consola.
En la terminal de Visual Studio Code, verificar que estamos dentro de la carpeta principal de nuestra aplicación y ejecutar el comando para levantar el servidor: ng serve -o
NOTA: En caso de mostrar un error que indica que la ejecución de scripts está deshabilitada, realizar lo siguiente:

  1. Abrir como administrador la aplicación "Windows PowerShell"
  2. Escribir el siguiente comando: Set-ExecutionPolicy Unrestricted
  3. Aceptar todos los mensajes que muestre
  4. Comprobar que los valores ya fueron modificados, al escribir: Get-ExecutionPolicy debemos obtener como respuesta Unrestricted

Trabajar con proyecto descargado de Github o ya existente (recrear carpeta node_modules)

En la terminal, posicionarse dentro de la carpeta del proyecto y ejecutar el siguiente comando: npm install
